# Ignofier

Create a gitignore file easily

## Installing

```
$ npm install -g ignofier
```

## Demo

![Gif Image](https://raw.githubusercontent.com/sujang958/ignofier/main/using.gif)

## Usage

_Create a .gitignore file_

```sh
$ ignofier
```

_Check version_

```sh
$ ignofier -v
```

_Update the gitignore repository_

```sh
$ ignofier -u
```
